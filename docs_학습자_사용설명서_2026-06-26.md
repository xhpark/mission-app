# mission_app 학습자 사용설명서

작성일: 2026-06-26
대상: 태국어 선교 학습 앱(mission_app)을 처음 사용하는 학습자

---

## 0. 시작하기 전에

- 이 앱은 **태국어 인사·감사·자기소개·축복 등 선교 현장에서 바로 쓸 수 있는 문장**을 매일 조금씩 배우는 앱입니다.
- 휴대폰에 인터넷(Wi-Fi 또는 데이터)이 연결되어 있어야 합니다. 로그인, 학습 진행 저장, 리포트 제출이 모두 서버와 통신합니다.
- 관리자(인솔자)의 **승인**이 있어야 학습을 시작할 수 있습니다. 가입만으로는 바로 학습할 수 없습니다.

---

## 1. 앱 설치하기

1. 관리자가 보내준 설치 파일(.apk) 링크를 받습니다.
2. 링크를 눌러 파일을 내려받습니다.
3. 내려받은 파일을 눌러 설치를 시작합니다.
   - "출처를 알 수 없는 앱은 설치할 수 없습니다" 같은 경고가 뜨면, **설정 → 허용**을 눌러 이번 설치만 허용해 주세요. (이미 알려진 발신자이므로 안전합니다.)
4. 설치가 끝나면 홈 화면에서 **mission_app** 아이콘을 찾아 실행합니다.

> 아이콘 이름이 "mission_app"으로 보이는 것이 정상입니다. 태국어 이름이 아니어도 맞는 앱입니다.

---

## 2. 계정 만들기 (최초 1회)

1. 앱을 실행하면 **학습자 로그인** 화면이 나옵니다.
2. 아직 계정이 없다면 **계정 만들기**를 누릅니다.
3. 다음 항목을 입력합니다.
   - **이름**
   - **전화번호** (숫자만, 8자리 이상)
   - **이메일**
   - **비밀번호** (6자 이상)
4. **계정 만들기** 버튼을 누르면 계정이 즉시 만들어지고 자동으로 로그인됩니다.

> 이름과 전화번호는 **관리자 승인**과, 나중에 비밀번호를 잊었을 때 **계정(이메일) 찾기**에 사용되니 정확히 입력해 주세요.

### 자주 발생하는 입력 오류
| 메시지 | 원인 / 해결 |
|---|---|
| 이메일과 비밀번호를 입력해 주세요 | 둘 중 하나라도 빈칸 |
| 비밀번호는 6자 이상이어야 합니다 | 비밀번호를 더 길게 |
| 계정 만들기에는 이름과 전화번호가 필요합니다 | 이름/전화번호 빈칸 |
| 전화번호를 정확히 입력해 주세요 | 숫자만, 8자리 이상으로 입력 |
| 이미 만들어진 계정입니다 | 이미 가입됨 → "로그인"으로 진행 |

---

## 3. 관리자 승인 기다리기

- 계정을 만들면 곧바로 **승인 대기** 화면이 나타납니다.
  - 화면 문구: "관리자 승인 대기 중입니다. 승인 후 학습을 시작할 수 있습니다."
- 이 화면에서 할 수 있는 일:
  - **상태 새로고침**: 관리자가 승인했는지 다시 확인
  - **로그인으로 돌아가기**: 일단 로그아웃하고 나중에 다시 로그인
- 관리자가 승인하면(관리자 대시보드에서 **승인** 버튼 클릭), 다음 로그인 또는 새로고침 시 바로 학습 선택 화면으로 들어갈 수 있습니다.

### 화면이 더 안 넘어갈 때 (참고)
| 화면 문구 | 의미 | 조치 |
|---|---|---|
| 이용 제한 / 이 계정은 사용할 수 없습니다 | 계정이 정지됨 | 관리자에게 문의 (로그인으로 돌아가기만 가능) |
| 학습 시작 제한 / 관리자 승인 후 학습을 시작할 수 있습니다 | 승인은 됐지만 학습이 일시적으로 막힘(예: 리포트 미제출 등) | 상태 새로고침, 안 풀리면 관리자에게 문의 |

---

## 4. 로그인하기

1. **학습자 로그인** 화면에서 이메일/비밀번호를 입력합니다.
2. **이메일 저장**을 체크하면 다음에 이메일을 다시 안 입력해도 됩니다.
3. **로그인** 버튼을 누릅니다.

### 비밀번호를 잊었다면
1. 로그인 화면에서 **이메일/비밀번호를 잊으셨나요?** 를 누릅니다.
2. **비밀번호 재설정**: 가입한 이메일을 입력하고 누르면 재설정 메일이 발송됩니다.
3. **이메일을 잊은 경우**: 가입 시 입력한 **이름**과 **전화번호**로 등록된 이메일을 찾을 수 있습니다.

---

## 5. 학습 선택 화면 둘러보기

로그인(+승인)이 끝나면 **학습 선택** 화면이 나옵니다. 이 화면이 앱의 "홈" 화면입니다. 위에서부터 순서대로:

### 5-1. 오늘의 학습과 복습 (가장 위, 따뜻한 색 카드)
- 관리자가 매일 새로 등록해 두는 **그날의 추가 학습/복습 자료** 링크입니다.
- **오늘의 링크 열기**를 누르면 외부 브라우저로 그날의 자료(웹페이지)가 열립니다.
- 오늘 등록된 자료가 없으면 "오늘 등록된 학습 링크가 없습니다" 안내가 나옵니다. → 정상이니 기다리거나 관리자에게 확인하세요.
- **이 링크 열기는 별도 학습이며, 아래 "학습 시작" 진도·리포트와는 무관합니다.** 안 눌러도 정규 학습엔 영향 없습니다.

### 5-2. 학습 선택 (일상 회화 / 선교 언어)
- 오늘 배우고 싶은 주제를 고릅니다.
  - **일상 회화**: 인사, 감사, 자기소개 등 생활 회화
  - **선교 언어**: 선교 현장에서 쓰는 표현

### 5-3. 난이도 선택 (초급 / 중급 / 고급)
- 본인 수준에 맞는 난이도를 고릅니다. 모르면 **초급**으로 시작하세요.

### 5-4. 학습모드 선택
- **문장 학습**: 새 문장을 보고 듣고 따라 말하며 배우는 단계
- **문장 테스트**: 배운 문장을 직접 맞혀보는 시험 단계
- **플래시 단어 학습 / 플래시 단어 테스트**: 단어 위주로 빠르게 익히고 테스트
- **플래시 문장 학습 / 플래시 문장 테스트**: 문장 위주로 빠르게 익히고 테스트

> 처음에는 "문장 학습"부터 시작해서, 다 배운 뒤 "문장 테스트"로 넘어가는 순서를 추천합니다.

### 5-5. 음성 인식 선택
- **서버 우선**: 서버에서 발음을 채점(더 정확하지만 인터넷 필요)
- **폰 전용**: 휴대폰 안에서만 채점(인터넷이 약해도 가능)
- 모르면 기본값(서버 우선) 그대로 두면 됩니다.

### 5-6. 시작 방식 선택
- **이어서 학습**: 전에 하던 학습을 이어서
- **새 학습 시작**: 처음부터 새로
- 진행 중인 학습이 없으면 자동으로 "새 학습 시작"이 적용됩니다.

### 5-7. 학습 시작 버튼
- 화면 맨 아래 **학습 시작**을 누르면 선택한 내용대로 학습이 시작됩니다.

---

## 6. 문장/단어 학습 진행하기

1. 화면에 태국어 문장(또는 단어)과 발음, 뜻이 나옵니다.
2. **듣기** 버튼으로 발음을 들어봅니다.
3. 소리 내어 따라 말해봅니다.
4. **다음**을 눌러 다음 문항으로 넘어갑니다.
5. 화면 위쪽에 작은 빨간 글씨로 "일상 회화, 초급, 문장 학습" 같은 현재 선택 정보가 표시됩니다 — 참고용이니 누르지 않아도 됩니다.
6. 학습을 마치면 자동으로 테스트 단계 또는 요약 화면으로 안내됩니다.

### 플래시 단어/문장 학습을 마친 후
학습을 끝내면 **테스트 선택 화면**이 나오고 3가지 중 고를 수 있습니다.
- **선택형 테스트 시작**: 보기 중에서 정답 고르기
- **말하기 테스트 시작**: 직접 말해서 발음 평가받기
- **플래시 단어(문장)학습 다시하기**: 학습을 한 번 더

---

## 7. 테스트 응시하기

- **선택형 테스트**: 제시된 한국어 뜻에 맞는 태국어 문장(또는 단어)을 보기 중에서 고릅니다. 제한 시간이 있으니 빠르게 고르세요.
- **말하기 테스트**: 제시된 뜻에 맞는 태국어 발음을 직접 말합니다. 마이크 녹음 버튼을 눌러 말하고, 평가 결과(유사도)를 확인합니다.
- 테스트 중 화면 상단 뒤로가기(◀)는 "한 문제 전으로", 화면 하단 "중지"는 "테스트를 그만두고 학습 선택으로 돌아가기"라는 점이 다릅니다. 실수로 누르지 않게 주의하세요.

---

## 8. 학습 요약 확인 → 리포트 제출하기

1. 학습/테스트를 모두 마치면 **학습 요약** 화면이 나옵니다. 총 문항 수, 정답률, 말하기 평균 유사도 등을 볼 수 있습니다.
2. 화면 아래 버튼:
   - **처음부터 다시 학습**: 학습 선택 화면으로 돌아가기
   - **다른 테스트 선택**: (해당하는 경우) 방금 했던 테스트의 다른 옵션(선택형/말하기/다시학습)으로 돌아가기
   - **리포트 제출하기**: 오늘 학습 결과를 관리자에게 보고
3. **리포트 제출** 화면에서 제출 전 마지막으로 학습 통계를 확인할 수 있습니다.
4. **리포트 제출** 버튼을 누르면 잠깐 "리포트 제출 중..."이 표시되고, 끝나면 "리포트를 제출했습니다" 메시지가 뜹니다.
5. 제출 후에는:
   - **학습 선택으로 이동**: 홈 화면으로
   - **다른 테스트 선택**: (해당하는 경우) 같은 학습모드의 다른 테스트 옵션으로 바로 이동

> 리포트를 제출해야 그날 학습이 정상적으로 기록됩니다. 학습만 하고 리포트를 제출하지 않으면 관리자가 학습 여부를 확인하기 어렵습니다. **반드시 끝까지 제출해 주세요.**

---

## 9. 이어서 학습하기 (앱을 끄고 다시 들어왔을 때)

- 학습 중간에 앱을 닫아도 진행 상황은 저장됩니다.
- 다시 앱을 열면 **이어서 학습** 화면이 나올 수 있습니다.
  - 진행 중인 세션이 있으면: 최근 세션 정보(주제/난이도/모드, 시작 시각)를 보여주고 **이어서 학습** 버튼으로 그 지점부터 계속할 수 있습니다.
  - 진행 중인 세션이 없으면: **선택 화면으로 이동** 버튼만 보이며 새로 시작합니다.
- **새 세션 시작**을 누르면 지금 진행 중이던 학습은 버려지고 학습 선택 화면부터 새로 시작합니다(되돌릴 수 없으니 신중히 누르세요).

---

## 10. 매일 학습 체크리스트 (요약)

1. 앱 실행 → 로그인
2. (맨 위) **오늘의 학습과 복습** → 오늘의 링크 열기로 추가 자료 확인
3. 학습 선택 → 난이도 선택 → 학습모드 선택 → **학습 시작**
4. 문장/단어를 따라 배우기
5. 이어지는 테스트(선택형 → 말하기) 응시
6. 학습 요약 확인 → **리포트 제출**
7. "리포트를 제출했습니다" 메시지를 확인하고 종료

---

## 11. 문제가 생겼을 때

| 증상 | 확인할 것 |
|---|---|
| 로그인이 안 됨 | 이메일/비밀번호 다시 확인, 인터넷 연결 확인 |
| "승인 대기"에서 안 넘어감 | 관리자에게 승인 요청, 화면에서 "상태 새로고침" |
| "이용 제한" / "학습 시작 제한" 문구가 보임 | 관리자에게 문의 |
| 오늘의 링크가 안 열림 | "오늘 등록된 학습 링크가 없습니다" 메시지면 정상(아직 미등록), 그 외 오류면 인터넷 연결 확인 후 재시도 |
| 리포트 제출이 안 됨 | 인터넷 연결 확인 후 "리포트 제출" 버튼 다시 누르기. 실패 시 안내 메시지("리포트 제출에 실패했습니다...")가 뜨니 잠시 후 재시도 |
| 말하기 테스트 평가가 이상함 | 음성 인식 선택을 "서버 우선"으로 바꿔서 재시도 |

---

문의: 관리자(인솔자)에게 직접 연락해 주세요.
