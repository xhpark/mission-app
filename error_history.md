# 오류 이력

## 2026-05-26 문장 테스트-선택형 보기 선택 후 발음 미재생

- 상태: 수정 및 회귀 테스트 추가
- 영향: 문장 테스트/플래시 문장 테스트 선택형에서 보기를 선택해도 오디오가 재생되지 않거나 잘못된 문장 오디오가 재생될 수 있음
- 원인: 선택지가 문장 ID와 오디오 경로 없이 태국어 문자열만 보유했고, 화면에서 문자열 역매칭으로 오디오를 찾았음. 중복 문장, 표기 차이, 빈 경로가 발생하면 오디오 재생이 조용히 무시될 수 있었음
- 조치: `sentenceThaiOptions`가 보기별 `sentenceId`와 `audioPath`를 함께 반환하도록 변경하고, 선택형 화면은 선택지 생성 시 확정된 오디오 경로만 사용하도록 수정함. 경로가 없거나 재생 실패 시 사용자에게 실패 메시지를 표시함
- 재발 방지: 보기별 ID/오디오 경로 존재 및 실제 에셋 존재를 검증하는 contract test를 추가함
