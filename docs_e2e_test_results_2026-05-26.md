## 남은 필수 수동 E2E

1. 실기기 release 또는 profile 빌드에서 앱 시작 후 로그인 화면으로 정상 진입하는지 확인한다.
2. QA 계정과 실사용 계정으로 로그인, 승인 상태, 학습 선택 진입을 각각 확인한다.
3. 문장/단어/플래시 단어/플래시 문장 학습을 각각 1회 완료하고 세션 요약으로 이동하는지 확인한다.
4. 선택형 테스트에서 보기를 선택했을 때 해당 보기 오디오가 끝난 뒤 다음 문항으로 이동하는지 확인한다.
5. 말하기 테스트에서 녹음 종료 후 유사도 확인이 표시되고, 확인 완료 전 자동 이동하지 않는지 확인한다.
6. 리포트 제출 화면의 수치가 세션 요약 수치와 일치하는지 확인한다.
7. 관리자 대시보드에서 승인 대기자, 리포트 현황, 일간/주간 통계가 실제 Firebase 데이터와 일치하는지 확인한다.
8. 앱을 백그라운드로 보냈다가 돌아왔을 때 로그인/학습 상태가 튕기지 않는지 확인한다.
